Executive SummaryCerebro v2 is a complete UI rewrite adopting the Ashisoft Duplicate File Finder layout paradigm: a single persistent window with toolbar, scan mode tabs, folder panel, grouped results list, inline preview, selection assistant, and status bar. The multi-page navigation system (Welcome, Mission, Scan, Review, Settings, History, Diagnostics, Themes — 9 pages total) is scrapped entirely.The backend uses a pluggable engine architecture where each scan type (files, images, videos, music, empty folders, large files) is an independent module implementing a shared interface. A scan orchestrator dispatches to the correct engine based on the active tab. A SQLite-backed hash cache provides Czkawka-level re-scan performance.### Scan modes (6 total)| Mode | Engine | Detection method | Key library ||------|--------|-----------------|-------------|| Files | `file_dedup_engine` | SHA256 / Blake3 byte-level | `hashlib`, `blake3` || Photos | `image_dedup_engine` | Perceptual hash (pHash + dHash) | `imagehash`, `Pillow` || Videos | `video_dedup_engine` | Keyframe extraction + pHash | `ffmpeg` (subprocess), `imagehash` || Music | `music_dedup_engine` | ID3 tag fuzzy match + duration | `mutagen` || Empty Folders | `empty_folder_engine` | Recursive `os.walk` bottom-up | stdlib only || Large Files | `large_file_engine` | Size-ranked file listing | stdlib only |### Ashisoft features replicated- Single-window, everything-visible layout (no page navigation)- Scan mode tab strip (Files / Photos / Videos / Music / Empty Folders / Large Files)- Folder tree panel with add/remove and drag-drop- Protect Folders — exclude directories from deletion- Grouped results list with checkboxes and sortable columns- Results sub-filtering by file type after scan (All / Images / Videos / Docs / Audio)- Side-by-side preview panel with synchronized zoom/pan- Selection Assistant with 8+ auto-mark rules- Status bar with live scan metrics- Settings as a popup dialog (not a page)- Recycle Bin deletion via `send2trash`- Auto-select by largest file / highest resolution---## 2. Engine Architecture — Pluggable Backend### Engine interface contract```python# cerebro/engines/base_engine.pyfrom abc import ABC, abstractmethodfrom dataclasses import dataclassfrom enum import Enumfrom typing import Callable, List, Optionalfrom pathlib import Pathclass ScanState(Enum):    IDLE = "idle"    SCANNING = "scanning"    PAUSED = "paused"    COMPLETED = "completed"    CANCELLED = "cancelled"    ERROR = "error"@dataclassclass ScanProgress:    state: ScanState    files_scanned: int    files_total: int           # 0 if unknown    duplicates_found: int    groups_found: int    bytes_reclaimable: int    elapsed_seconds: float    current_file: str          # path of file currently being processed@dataclassclass DuplicateGroup:    group_id: int    files: List["DuplicateFile"]    total_size: int            # sum of all file sizes in group    reclaimable: int           # total_size minus the kept file@dataclassclass DuplicateFile:    path: Path    size: int    modified: float            # timestamp    extension: str    is_keeper: bool            # True = auto-selected to keep    similarity: float          # 0.0-1.0, 1.0 = exact match    metadata: dict             # engine-specific (resolution, bitrate, etc.)class BaseEngine(ABC):    """Interface that all scan engines must implement."""    @abstractmethod    def configure(self, folders: List[Path], protected: List[Path],                  options: dict) -> None:        """Set scan parameters before starting."""    @abstractmethod    def start(self, progress_callback: Callable[[ScanProgress], None]) -> None:        """Begin scanning in a background thread. Non-blocking."""    @abstractmethod    def pause(self) -> None:        """Pause the current scan. Resumable."""    @abstractmethod    def resume(self) -> None:        """Resume a paused scan."""    @abstractmethod    def cancel(self) -> None:        """Cancel the current scan. Not resumable."""    @abstractmethod    def get_results(self) -> List[DuplicateGroup]:        """Return grouped results after scan completes."""    @abstractmethod    def get_progress(self) -> ScanProgress:        """Return current progress snapshot."""    @abstractmethod    def get_mode_options(self) -> List[dict]:        """Return list of configurable options for left panel.        Each dict: {name, type, default, choices, tooltip}        """```### Engine orchestrator```python# cerebro/engines/orchestrator.pyclass ScanOrchestrator:    """Dispatches to the correct engine based on active scan mode."""    def __init__(self):        self._engines = {            "files":         FileDedup Engine(),            "photos":        ImageDedupEngine(),            "videos":        VideoDedupEngine(),            "music":         MusicDedupEngine(),            "empty_folders": EmptyFolderEngine(),            "large_files":   LargeFileEngine(),        }        self._active_engine: Optional[BaseEngine] = None        self._active_mode: str = "files"    def set_mode(self, mode: str) -> List[dict]:        """Switch active engine. This is a view filter on the in-memory results list.### Scan pipeline detail (Files mode)```Input: List of folder paths + options (hash_algo, min_size, max_size, extensions)  │  ├─ 1. Walk directories (ThreadPool) → collect file metadata (path, size, mtime)  │     Filter by: min/max size, extension whitelist/blacklist, skip protected paths  │  ├─ 2.
